Arquitetura da rede neural física

O modelo combina uma Physics-Informed Neural Network com integração numérica RK4:

→ Lei do gás ideal para pressão de câmara via CO₂ gerado pela reação

→ Tsiolkovsky com Isp real: v_ej calculado por Bernoulli com Cd_nozzle = 0.62

→ Arrasto aerodinâmico com Cd configurável e área frontal do CORPO do foguete

→ Pressão de selamento: impulso só começa quando P_câmara ≥ P_selo

→ Perda de arrasto durante fase de impulso calculada analiticamente

→ Integrador RK4 em passos de 5ms + sweep refinado a 0.5°

Equações físicas
// Reação: NaHCO₃ + CH₃COOH → CO₂ + H₂O + sal n_CO₂ = (m_prop · frac_bicarb) / 0.084 mol // Lei do gás ideal → pressão de câmara P_câmara = P_atm + n_CO₂ · R · T / V_câmara [Pa] // Bernoulli com coef. descarga do bocal (Cd_nozzle = 0.62) v_ej = 0.62 · √(2 · (P_câmara - P_selo) / ρ) [m/s] // Fluxo de massa e tempo de ejeção ṁ = ρ · A_nozzle · v_ej [kg/s] t_ej = m_prop / ṁ [s] // Tsiolkovsky → ΔV ΔV = v_ej · ln(m₀ / m_final) [m/s] // V₀ real (subtrai perda de arrasto durante impulso) V₀ = ΔV - (F_drag_médio / m) · t_ej [m/s] // Trajetória — RK4 por passo de 5ms ρ_ar = P_atm / (287.05 · T) [kg/m³] A_frontal = π · (d_corpo/2)² [m²] ← área do CORPO F_drag = 0.5 · ρ_ar · Cd · A_frontal · v² [N]
